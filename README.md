# Danfoss ECL 310 for Home Assistant

This integration connects your Danfoss ECL 310 (A247) heating controller to Home Assistant using Modbus TCP.

## Features
- Full support for Home Assistant 2025.12+ (Pymodbus 3.8+)
- Climate entity for temperature control
- Temperature sensors (S1-S6)
- Diagnostic sensors for pumps and valves
- German and English translations

## Installation
1. Copy the `danfoss_ecl310` folder to your `custom_components` directory.
2. Restart Home Assistant.
3. Add the integration via the UI (Settings -> Devices & Services).
